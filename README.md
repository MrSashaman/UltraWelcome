# UltraWelcome — SCP:SL LabAPI Plugin

A plugin for **SCP: Secret Laboratory (LabAPI)** This will add welcome text to your server.
----
## Features:
- 🪙 BroadCast Welcome
- 🌲 Hint Text Welcome
- 👋 Customizable welcome messages for players 
- 🔥 Friendly Fire enabled at the end of the round  

---

## Installation:

### 1. Download the latest version of the plugin from:
```bash
https://github.com/MrSashaman/UltraWelcome/releases
```
### 2. Move the file to the specified path:
```bash
C:\Users\AppData\Roaming\SCP Secret Laboratory\LabAPI\plugins\global
```
### 3. Run server
